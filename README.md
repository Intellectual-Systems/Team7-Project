## How to Run

1. Go to [colab.research.google.com](https://colab.research.google.com)
2. `File → Open notebook → GitHub` → paste your repo URL
3. Open `press_start.ipynb`
4. Run cells top to bottom — sections are clearly labelled

## Git Workflow
- Each member works on the notebook sequentially to avoid merge conflicts
- Branch per feature: `git checkout -b feature/preprocessing`
- One logical change per commit — atomic commits
- Never commit `data/`, plots, or `kaggle.json`
